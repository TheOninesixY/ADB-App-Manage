# Py ADB Manage
## 不止管理。
### 超强应用管理
- 查看已安装应用。
- 搜索应用。
- 卸载应用。
- 安装应用。

### 内置文件管理器
- 替代难用的adb命令，实现图形化管理。
- 一键安装APK文件。

### 内置ADB
- 无需配置环境变量，直接使用。

### 精美的UI
- 仿制微软流畅设计体系，肥肠美观。（其实是有点丑的）

---

### 碎碎念
我一开始开发这个应用的原因纯是因为WSA的应用管理真的做的巨烂，在该应用刚开发出来时它只有安装应用的功能。后来需求不断增加，于是又有了应用卸载、查看当前安装的应用、文件管理器等功能。

我在开发这个应用时其实还试过很多次用HTML来当图形化，第一次是直接开本地服务器用浏览器访问，但后来觉得这样不优雅，所以不搞了。第二次是Electron，结果端口问题太难解决了，所以放弃了。第三次是PyWebView，结果PyWebView死活装不上，pip还炸了，还没开始就结束了。
